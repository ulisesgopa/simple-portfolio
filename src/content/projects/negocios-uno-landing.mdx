## Overview
**Negocios Uno** is a platform that helps small and medium businesses manage their operations more efficiently.

For its online presence, I developed the **official landing page**, focusing on presenting the main features of the platform with a modern and easy-to-navigate design. The site was built using **Astro** and **TailwindCSS**, optimized for performance and responsive on all devices.

## ‚ú® Solution

### Tech Stack
- **Astro:** For fast and optimized static site generation.
- **TailwindCSS:** For a clean and responsive design.
- **Mobile-First Approach:** Designed to work well on phones, tablets, and desktops.

### Key Features
1. **Clear Sections:** Easy to read and navigate.
2. **Responsive Design:** Good experience on any screen size.
3. **Smooth Animations:** Enhancing user interaction without slowing down.
4. **Fast Loading:** Optimized code with Astro for performance.

1. **Landing:** Users see the platform's mission.
2. **Features:** Discover how Negocios Uno can help their business.
3. **Call to Action:** Buttons to get started or contact the team.
